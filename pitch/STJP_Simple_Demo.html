How it works

Five steps from plain English to guaranteed coordination.

1

You write an intent

“Produce a quarterly finance report; high revenue needs a tax audit before it's approved.”

↓
2

An LLM drafts a formal protocolScribble-checked

The draft is run through a real session-type checker. Can it deadlock? Loop forever? If so it's rejected and redrafted — before any agent runs.

↓
3

It's projected into per-agent contracts

Each agent receives only its own lines: what it may send, what it must wait for, what values are valid.

↓
4

A tiny monitor sits beside each agent

Plain Python, not an AI. It checks every message against the contract — instantly, deterministically.

↓
5

The gate stops mistakes before they landenforced

A wrong or out-of-order message is rejected before delivery and the agent is asked to try again. Mistakes don't just get logged — they can't happen.
